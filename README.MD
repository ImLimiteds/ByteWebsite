## :toolbox: Getting Started

### :bangbang: Prerequisites

- Install Ruby<a href="https://www.ruby-lang.org/en/documentation/installation/"> Here</a>
- Install Sinatra
```bash
gem install sinatra
```


### :running: Run Locally

Clone the project

```bash
https://github.com/ImLimiteds/ByteWebsite
```
Go to the project directory
```bash
ruby app.rb
```
